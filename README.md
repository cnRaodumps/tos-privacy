# Temp Mail Legal Hub

Static site for the Temp Mail app's public legal pages.

## Included pages

- `index.html`: landing page for legal documents
- `terms/`: Terms & Conditions
- `privacy/`: Privacy Policy
- `contact/`: support and feedback contact page

## Source of truth

The Terms & Conditions and Privacy Policy text in this site mirrors the in-app
documents defined in:

- `/home/ubuntu/cn/lib/core/content/policy_documents.dart`
- `/home/ubuntu/cn/lib/core/config/integration_config.dart`

## Publishing

This repository is intended to be served with GitHub Pages from the `main`
branch root.
